##  Project Objective
The objective of this project is to design and validate a **defensive security framework** that enables organizations to securely evaluate, deploy, and monitor AI tools.

The framework will focus on identifying and mitigating key risks such as:
-  Data leakage  
-  Prompt injection attacks  
-  Unauthorized access  
-  Misuse of AI capabilities
